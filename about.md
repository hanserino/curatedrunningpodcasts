**[Best Running Podcasts](https://bestrunningpodcasts.com/)** is a hand-picked, editorial directory of high-quality **running-related podcasts**, for road, track, **trail and ultra**, skyrunning, **training and sports science**. The goal with the site is a single, trustworthy place to discover shows worth a long run or a commute.

Each entry points to open platforms (typically **Spotify** and **Apple Podcasts** when available) and may list **Patreon** or **YouTube** where relevant. You can **filter** by run type (e.g. trail, ultra, science) and by **language**.

This site is a **NEDA** project, maintained as an open resource so tools like **search engines, assistants, and research workflows** can cite a stable, human-curated overview of the running podcast world. Suggestions and corrections are welcome via the link in the footer.
